# Hydra – Fortress Tool #1

Hydra is the first tool in the Fortress ecosystem — a multi‑mode password analysis engine designed for debugging, security auditing, and developer workflows.

## Features
- Entropy scoring
- Pattern detection
- Mutation analysis
- Structural breakdown
- Character distribution
- JSON output mode
- Score‑only mode
- Suggestions‑only mode
- Full combined output mode

## Modes
1. **Full Report** – Complete breakdown of password structure and security
2. **JSON Output** – Developer‑friendly structured output
3. **Score Only** – Quick strength score
4. **Suggestions Only** – Improvement recommendations
5. **Everything Mode** – All data combined
